An interpreter for the lovely [BrainFu*k](https://esolangs.org/wiki/Brainfuck) language. Based on the tutorials in the [Beautiful Racket](https://beautifulracket.com) book.

++++++[>++++++++++++<-]>.
>++++++++++[>++++++++++<-]>+.
+++++++..+++.>++++[>+++++++++++<-]>.
<+++[>----<-]>.<<<<<+++[>+++++<-]>.
>>.+++.------.--------.>>+.

// Hello World!
